# mac-VIM
This is the vim .vimrc configuration for python IDE.




For mac, need several dependencies: 

pip install dbgp pep8 flake8 pyflakes isort\\




\\shortcuts:

F2 task:

F3: NERDTree

F4: tagbar

F5: change background

F6: change python interpreter

F9: run the code
